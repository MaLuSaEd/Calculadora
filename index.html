Calculadora de funciones

Bienvenidx a la calculadora de funciones, esta herramienta esta destinada a estudiantes que quieran dar sus primeros pasos en el analisis de funciones. ¿Qué es una función?, ¿qué significa el dominio de una funcion?, ¿cuál es la diferencia entre un minimo local y un minimo global?, ¿cuáles son los distintos tipos de conjuntos que me conviene definir para estudiar el comportamiento de una función? ¿cuál es la realcion entre una función y su derivada?

Estas son el tipo de preguntas que esperamos responder y generar una base para luego afrontar problemas mas difíciles.

Veamos algunas definiciones que usaremos para efectuar nuestro analisis:

  • Dominio = \(\{x\ \in\ \mathbb{R}:\ f(x)\ \in \mathbb{R} \}\), el conjunto de valores de x donde la funcion está definida.
  • \(C_{0} = \{x\ \in \mathbb{R}:\ f(x) = 0 \}\), el conjunto de valores de x donde la función vale 0.
  • \(C^{+} = \{x\ \in \mathbb{R}:\ f(x) > 0 \} \), el conjunto de valores de x donde la función es positiva.
  • \(C^{-} = \{x\ \in \mathbb{R}:\ f(x) < 0 \} \), el conjunto de valores de x donde la función es negativa.
